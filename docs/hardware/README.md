# Materials and Tools

| Material / Tool | Description |
|---|---|
| Pulsar H2/C6 | Main microcontroller platform used for wireless communication and processing |
| 3D Printer | Used to manufacture the enclosure and mechanical components |
| Soldering Kit | Required for electronic assembly and prototyping |
| QWIIC Connectors | Standardized modular connection system |
| DevLab BMI270 Sensor | Motion and orientation sensor |
| DevLab QWIIC Expander | Multi-device expansion interface |
| OLED SSD1306 Display | Display used for emotions and animations |
| LiPo Battery 3.7V | Portable rechargeable power source |
